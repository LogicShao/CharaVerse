# CharaVerse

OC（原创角色）管理系统与名片展示应用

## 技术栈

- **前端框架**：React 19 + TypeScript
- **构建工具**：Vite
- **状态管理**：Zustand
- **路由**：React Router
- **数据验证**：Zod
- **样式**：CSS Modules

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 类型检查
npm run build

# 代码检查
npm run lint
```
